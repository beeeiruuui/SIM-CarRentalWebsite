                        1. INTRODUCTION
================================================================================

This report presents the web design considerations and implementation details 
for the AZoom Car Rental website, developed as part of ISIT207 Frontend Web 
Programming Assignment 2.

AZoom Car Rental is a local electric car rental company that provides 
environmentally-friendly transportation options. The website aims to provide 
customers with an intuitive platform to browse available vehicles, make 
reservations, and manage their bookings online.

The website supports the following business processes:
- Customer car reservation via credit card only (no cash/PayPal)
- Vehicle browsing and filtering (9 electric/hybrid vehicles from 3 brands)
- Flexible rental periods: Daily, Weekly (10% discount), Monthly (20% discount)
- Two branch locations for pickup/return: Downtown (123 Electric Ave) and Airport (456 Terminal Rd)
- User account management (customer and staff)
- My Bookings page for viewing booking history
- Payment processing with credit card validation
- Admin dashboard for staff management

                    3. WEB DESIGN CONSIDERATIONS
================================================================================

3.1 VISUAL DESIGN
-----------------
Color Scheme:
- Primary: Dark navy blue (#1a1a2e, #16213e) - Conveys professionalism and trust
- Accent: Bright blue (#007bff) - For call-to-action buttons
- Background: Light gray (#f8f9fa) - Clean, modern appearance
- Text: Dark gray (#333) - Excellent readability

Typography:
- Font Family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- Clean, modern sans-serif fonts for readability
- Hierarchical sizing for headings (h1 > h2 > h3)
- Line height of 1.6 for comfortable reading

3.2 LAYOUT DESIGN
-----------------
- Fixed header for consistent navigation across all pages
- Maximum width container (1200px) for optimal reading
- CSS Grid and Flexbox for responsive layouts
- Consistent spacing using CSS clamp() for fluid design
- Card-based design for car listings and information display
- Consistent footer across all pages with three columns:
  * For Renters: Book Now, My Bookings
  * Company: About Us, Our Services, Our Branches
  * Our Branches: Downtown (123 Electric Ave), Airport (456 Terminal Rd), 
    Operating hours (Mon-Sun 8AM - 8PM)
- Social media links in footer (Facebook, Instagram, Twitter)

3.3 RESPONSIVE DESIGN
---------------------
The website is fully responsive and adapts to:
- Desktop screens (1200px+)
- Tablets (768px - 1199px)
- Mobile devices (below 768px)

Key responsive features:
- Hamburger menu for mobile navigation
- Flexible grid layouts that stack on smaller screens
- Fluid typography using clamp() function
- Touch-friendly button sizes on mobile
